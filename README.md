# Global-Economic-Analysis-PCA

Acest proiect realizează o analiză statistică asupra economiei mondiale, utilizând date reale de la Banca Mondială pentru 217 state (anul de referință 2022). Scopul a fost simplificarea a 7 indicatori complecși (PIB, inflație, șomaj, sănătate, digitalizare) pentru a vizualiza tiparele de dezvoltare globală.

Prin aplicarea algoritmului PCA (Principal Component Analysis) în Python, am redus dimensionalitatea datelor și am demonstrat că economia lumii se împarte în două direcții majore: Nivelul de Dezvoltare Socio-Economică (bogați vs. săraci) și Instabilitatea Macroeconomică (economii stabile vs. economii în criză). Studiul evidențiază polarizarea Nord-Sud și identifică statele vulnerabile la șocuri inflaționiste.
